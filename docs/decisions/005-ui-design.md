# ADR-005: UI 디자인 방향

## 상태
진행 중 (2026-02-25)

## 벤치마킹
- whiplace.com을 주요 참고 (키워드순위 추적 대시보드)
- 차별화: 포인트 컬러, 레이아웃, 순위 표시 방식 변형

## 핵심 UI 요소 (whiplace 참고)

### 대시보드
- 매장 카드 + 키워드별 순위 행
- 큰 순위 숫자 (눈에 띄는 표시)
- 방문자/블로그 리뷰 수 뱃지
- 전체 수집 버튼, 자동 추적 토글

### 키워드 추가 모달
- 매장 선택 드롭다운
- 키워드 입력
- 수집 주기/시간 설정
- 순위 알림 토글
- 팁: "15시경에 확정" 안내

### 차별화 포인트
- 포인트 컬러: 초록 대신 파란/남색 계열
- 순위 표시: 원형 뱃지 등 독자적 스타일
- 카드/버튼 배치 변형
